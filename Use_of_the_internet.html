Communication

 Voice

  • Audio Conferencing - The connection requires 2 computers using microphones and speakers. Examples of software that can be used for this includes Skype and Teams.

  • Video conferencing - This is much the same as audio conferencing except webcams are used and users can see each other.  In some cases, for example, Cisco, Teams and Zoom, more than two people can join at the one time. Video conferencing can be used on computers, smartphone or tablets.
  • VOIP - phone calls over the internet. Can be used on computer, VOIP phone or smart phone. The receiver of the call doesn't have to be using VOIP. Examples of VOIP apps include Skype, Blueface and Speechpath. Advantages of using VOIP include the ability to have 3-way conversations and to share files while on the call.

Written communication

  • Email - transmission of messages over a network. Emails can be in the form of files or written messages. Some mail systems are confined to a single network such as a business organisation while other systems allow you to send messages all over the world. Provided you have another person’s email address, you can send email. Software used Outlook or Eudora. Advantages of email: they only takes a few seconds to send, they can be sent to a group of people at once, you can set time when email sent, they are a very effective way to advertise. A disadvantage of emails is that spam or junk mail can spread Trojans and viruses. Therefore, it is important to always use updated anti-virus.
  • Instant messaging - This is real time messaging over the internet. Provided the recipient is online, the message appears instantly. It is text based conferencing between two or more people. It eliminates the need for phone calls and is very convenient for group messages. It can be for personal use, for organisations and businesses and is an inexpensive or free form of messaging.
  • IRC - Internet relay chat. People can meet in chat rooms that are grouped based on topic or meet privately. An IRC client, for example Slack, and internet access is required. People can chat, share files. Private Chatrooms can be created which are password protected. A disadvantage with IRC is a possible lag or delay in messages being sent or received, depending on individual internet connections.


  • SMS – An sms can be no longer than 160 characters and contains no images or graphics. Once a message is sent, it received by a short message service centre (SMSC) which then relays it to the appropriate mobile device. It has evolved to MMS (multimedia messaging service) which can include text, sounds, images and video. This is provided the recipient has an MMS capable device. Examples include WhatsApp and Messenger.

Wireless communications

  • Mobile phones - With the use of mobile broadband, phone calls, emails, sms messages, mms messages and video calls can now all be accessed on smart mobile devices provided the internet signal is powerful enough.


Collaboration

File sharing - involves sharing files among several people within a network for the purposes of sharing information or collaboration on a piece of work. The files may be copied, printed or modified depending on the level of access the user has.
Peer-to-peer file sharing - books, movies, music or games can be shared among a group of network users with equal status. Bits of files can be shared from different users, provided the computer is turned on, bypassing copyright laws. File sharing software includes Napster, Bit Torrent and The Pirate Bay.
File sync and sharing - can be done using services such as One Drive and Dropbox. These are cloud based. Any changes to the files are automatically updated on each user’s device. They can be easily accessed using mobile devices, allowing people to view or collaborate on files from any location. 

Research

The World Wide Web - contains a vast amount of content in the form of web pages that contain text, videos and images. They are formatted in HTML and connected with each other using hyperlinks. Once a device has an internet connection, this information can be searched and retrieved using browsers and search engines.
Browsers - computer software programs that are used to explore, retrieve and display content from the World Wide Web. Content can include web pages, videos, images and other files. The browser contacts the web server to request information and the web server sends back that information to the browser, which then translates the HTML into texts, images etc. to be displayed on the user’s device. Examples of browsers include Firefox, developed by Mozilla, Microsoft edge, and Safari, which is a browser for apple products.
Search engines - We use search engines to search the World Wide Web for specific information. We do this by entering keywords or phrases into the search engine. The search engine then systematically explores the web and returns a list of results that are the best match, based on these keywords. Examples of search engines include Google, Bing and DuckDuckGo.



E-Commerce

E-commerce relates to the buying and selling of goods online to internet users. It has grown massively in the last decade alongside advancements in internet infrastructure, smart devices and online payment capabilities. It can relate to:

cla
  • B2C business - such as Amazon and eBay.

  • C2B business - an example being selling photos to Shutterstock.

  • B2B business - This can often be in the form of services or software. An example would be IBM offering cloud computing to businesses.

  • C2C business - Done Deal is an example where customers can sell goods privately to other customers online.

The types of products sold online can be physical e.g. clothes from Littlewoods, services such as consultancy services (IBM’s Supply Chain Consulting would be an example of this), digital such as online books or courses, subscription based products such as online newspapers or Netflix or wholesale to vendor such as AliBaba.
Goods can be bought through a business’s own website. There is also the option of selling goods on ecommerce platforms like Shopify, (ideal for small businesses), Amazon or eBay.
